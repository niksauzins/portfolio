# Personal Portfolio — nickroger.dev

Minimalist, high-performance developer portfolio showcasing my projects, skills, and journey as a software developer.

## Tech Stack
- **Frontend:** HTML5, Tailwind CSS v4, JavaScript
- **CI/CD:** GitHub Actions (Automated build & deploy)
- **Design:** Mobile-first, Dark Mode, Minimalist UI
- **Hosting:** GitHub Pages
- **DNS/SEO:** Cloudflare, Google Search Console optimized

## Key Features
- **Tailwind v4 CLI:** Using the latest Tailwind version for an optimized build process.
- **Automated Workflows:** Every push to `main` triggers a GitHub Action that builds the CSS and deploys the site.
- **Fully Responsive:** Optimized for everything from small mobile screens to large desktops.
